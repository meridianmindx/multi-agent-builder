# Changelog

All notable changes to the multi-agent-builder project will be documented here.

## [1.0.0] - 2026-06-29

### Added
- CLI tool for scaffolding AI agent projects
- **Basic template** — Minimal agent with tool registry and config
- **Advanced template** — Full agent with tool registry, config, and main loop
- Multi-package builder with template-driven scaffolding
- Full project structure with `multi_agent_builder` package

### Notes
- Zero external dependencies required at runtime
- Install via `pip install multi-agent-builder` or from source
